Request Header Parser Microservice

Use the API endpoint /api/whoami to get your IP address, preferred language, and user-agent.

Example usage:

  • /api/whoami

Example output:

{ "ipaddress": "159.20.14.100", "language": "en-US,en;q=0.5", "software": "Mozilla/5.0 (Windows NT 6.1; WOW64; rv:40.0) Gecko/20100101 Firefox/40.1" }
